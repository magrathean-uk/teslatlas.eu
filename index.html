Trust

Small claims. Clear boundaries.

This website should make the trust model obvious before anyone taps download.

No cloud relay. Teslatlas connects from your device to the source you configure. Vehicle data does not pass through Magrathean by default.

No write-back path. Teslatlas reads from TeslaMate or the configured API source into a local mirror. It is not a Tesla command app and does not write back to TeslaMate by default.

No affiliation claim. Tesla, TeslaMate, and MyTeslaMate remain separate third-party brands or services. Teslatlas exists for descriptive interoperability only.

Local mirror SQLite cache, prepared map artifacts, and app preferences stay on-device.
Access details Connection details stay in platform-protected storage on the device.
Output limits Battery, charging, and efficiency outputs are informational only. Check the vehicle, Tesla, or qualified service records before acting on them.
